# Stock Screener

## Get started

1. Set Up Virtual Environment

   ```bash
   python3 -m venv virtualenv
   source virtualenv/bin/activate
   ```

2. Run Flask Backend

   ```bash
   flask run --host=0.0.0.0 --port=8081
   ```

3. Run Expo Go Frontend

   ```bash
   cd stock-app
   ```

   ```bash
   npx expo start
   ```
